## Description

My senior project at BYU-Idaho for my bachelors degree. 

## Dev

To setup a development environment for this app run

```
yarn install
yarn dev
```

in a separate terminal run 

```
npx nodemon .\server\server.js
```

## Build

To build and start the app run

```
yarn install
yarn build
yarn prod
```
